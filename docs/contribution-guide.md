# Contribution Guide

The atlas accepts case suggestions that document climate-responsive architectural, spatial, material, or body-scale strategies.

## Case submission template

### 1. Case identity

- Case title
- Regional variant name, if applicable
- Location
- Coordinates
- Climate zone
- Period
- Typology
- Material
- Scale
- Source category

### 2. Case interpretation

A short paragraph explaining why the case matters as climate-adaptation knowledge.

### 3. Climate pressures

List the main environmental pressures.

Examples:

- Heat stress
- Solar exposure
- Humidity
- Wind exposure
- Cold stress
- Rainfall
- Diurnal variability

### 4. Adaptive logic

Use the atlas hierarchy:

```text
Condition → Mechanism → Strategy Family → Regional Variant → Element
```

### 6. Media

Provide only images that can be used legally.

For each image, include:

- filename
- caption
- photographer/source
- URL/publication
- license or permission status

### 7. References

Include academic, archival, fieldwork, or other credible references.

### 8. Contributor information

- Name(s)
- Institution
- Semester or project context, if applicable
- Contact, optional
