# Case images

Create one folder per case, using the same slug as the atlas case ID or case title.

Example:

```text
assets/cases/mashrabiya/
assets/cases/jali-screen/
assets/cases/antakya-courtyard-house/
```

Each folder should contain:

```text
hero.jpg
diagram.jpg
plan.jpg
source.txt
```

Only include images that are intentionally selected and credited.
